# FilePath: removeExtension() static method

The **`removeExtension()`** static method of the `FilePath` object removes the extension from a path and returns the result.

## Syntax

```js
FilePath.removeExtension(path)
```

### Parameters

- `path`

    A string with the file path.

### Return value

A string without the extension.

## Description

**`removeExtension()`** removes the extension from a path and returns the result.
