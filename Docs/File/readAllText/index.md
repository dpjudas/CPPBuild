# File: readAllText() static method

The **`readAllText()`** static method of the `File` object
reads an entire file from disk and returns it as a string.

## Syntax

```js
File.readAllText(filename)
```

### Parameters

- `filename`

    A string with the path to the file.

### Return value

A string with the file contents.

## Description

**`readAllText()`** reads an entire file from disk and returns it as a string. The text encoding is UTF-8.
