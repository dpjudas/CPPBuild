# Targets

The **`Targets`** interface is used to create targets.

## Static methods

- [`Targets.addApplication()`](addApplication/index.md)

    Adds an application target (/subsystem:windows, app bundle).

- [`Targets.addConsole()`](addConsole/index.md)

    Adds a console target (/subsystem:console).

- [`Targets.addStaticLibrary()`](addStaticLibrary/index.md)

    Adds a static library target (.lib, .a).

- [`Targets.addDynamicLibrary()`](addDynamicLibrary/index.md)

    Adds a dynamic library target (.dll, .so, .dylib).

- [`Targets.addWebsite()`](addWebsite/index.md)

    Adds an emscripten website target (.webpkg).

- [`Targets.addWebComponent()`](addWebComponent/index.md)

    Adds an emscripten web component target.

- [`Targets.addWebLibrary()`](addWebLibrary/index.md)

    Adds an emscripten static library with CSS resources target.
