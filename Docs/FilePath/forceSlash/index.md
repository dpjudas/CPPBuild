# FilePath: forceSlash() static method

The **`forceSlash()`** static method of the `FilePath` object replaces all backslashes in a path with slashes and returns the result.

## Syntax

```js
FilePath.forceSlash(path)
```

### Parameters

- `path`

    A string with the file path.

### Return value

A string with no backslashes.

## Description

**`forceSlash()`** replaces all backslashes in a path with slashes and returns the result.
