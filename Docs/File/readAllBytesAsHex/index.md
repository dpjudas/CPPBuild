# File: readAllBytesAsHex() static method

The **`readAllBytesAsHex()`** static method of the `File` object
reads an entire file from disk and returns it as a hex string.

## Syntax

```js
File.readAllBytesAsHex(filename)
```

### Parameters

- `filename`

    A string with the path to the file.

### Return value

A hex string with the contents of the file.

## Description

**`readAllBytes()`** reads an entire file from disk and returns it
as a hex string. The string is lowercase.
