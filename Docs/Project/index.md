# Project

The **`Project`** interface is used to set up project global settings.

## Static methods

- [`Project.setName()`](setName/index.md)

    Sets the name of the project.

- [`Project.addConfiguration()`](addConfiguration/index.md)

    Adds a configuration.

- [`Project.addSubdirectory()`](addSubdirectory/index.md)

    Adds a subdirectory with an additional configure script.
