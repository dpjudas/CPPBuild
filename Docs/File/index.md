# File

The **`File`** interface provides access to reading or writing files on the disk.

## Static methods

- [`File.readAllText()`](readAllText/index.md)

    Read entire file as UTF-8 text and return it as a string.

- [`File.readAllJson()`](readAllJson/index.md)

    Read entire file as UTF-8 text and parse it as JSON.

- [`File.readAllBytes()`](readAllBytes/index.md)

    Read entire file and return it as a byte array.

- [`File.writeAllText()`](writeAllText/index.md)

    Write string to file as UTF-8 text.

- [`File.writeAllJson()`](writeAllJson/index.md)

    Write JSON stringify and write to file as UTF-8 text.

- [`File.writeAllBytes()`](writeAllBytes/index.md)

    Write byte array to file.
