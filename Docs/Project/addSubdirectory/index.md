# Project: addSubdirectory() static method

**`Project.addSubdirectory()`** adds a subdirectory with a configure script to run.

## Syntax

```js
Project.addSubdirectory(path)
```

### Parameters

- `path`

    A string with the path to the subdirectory. The path is relative to the current
    configure script. The script to run uses the same name as the subdirectory
    with `.js` appended.

### Return value

None.
