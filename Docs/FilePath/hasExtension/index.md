# FilePath: hasExtension() static method

The **`hasExtension()`** static method of the `FilePath` object checks if a file extension matches.

## Syntax

```js
FilePath.hasExtension(path, extension)
```

### Parameters

- `path`

    A string with the path to check.

- `extension`

    A string with the extension to check for.

### Return value

A boolean that is true if the extension matches.

## Description

**`hasExtension()`** checks if a file extension matches the given path.
