# PackageInstaller

The **`PackageInstaller`** interface is used to create packages.

## Static methods

- [`PackageInstaller.add()`](add/index.md)

    Adds a package installer.

## Instance methods

- [`PackageInstaller.addFiles()`](addFiles/index.md)

    Adds files to the package.

- [`PackageInstaller.addDefines()`](addDefines/index.md)

    Adds defines to targets using the package.

- [`PackageInstaller.addCompileOptions()`](addCompileOptions/index.md)

    Adds compile options to targets using the package.

- [`PackageInstaller.addLinkOptions()`](addLinkOptions/index.md)

    Adds linker options to targets using the package.

- [`PackageInstaller.addIncludePaths()`](addIncludePaths/index.md)

    Adds include paths to targets using the package.

- [`PackageInstaller.addLinkLibraries()`](addLinkLibraries/index.md)

    Adds libraries to link to targets using the package.

- [`PackageInstaller.addLibraryPaths()`](addLibraryPaths/index.md)

    Adds library paths to targets using the package.
