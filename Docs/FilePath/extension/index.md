# FilePath: extension() static method

The **`extension()`** static method of the `FilePath` object returns the file extension for a path.

## Syntax

```js
FilePath.extension(path)
```

### Parameters

- `path`

    A string with the file path.

### Return value

A string with the extension, if any.

## Description

**`extension()`** returns the file extension for a path.
