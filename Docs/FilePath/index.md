# FilePath

The **`FilePath`** interface simplifies working with file path strings.

## Static methods

- [`FilePath.hasExtension()`](hasExtension/index.md)

    Returns true if the file extension matches.

- [`FilePath.extension()`](extension/index.md)

    Returns the file extension.

- [`FilePath.removeExtension()`](removeExtension/index.md)

    Returns path with file extension removed.

- [`FilePath.firstComponent()`](firstComponent/index.md)

    Returns the first path component.

- [`FilePath.removeFirstComponent()`](removeFirstComponent/index.md)

    Returns path without the first component.

- [`FilePath.lastComponent()`](lastComponent/index.md)

    Returns the last path component.

- [`FilePath.removeLastComponent()`](removeLastComponent/index.md)

    Returns path without last component.

- [`FilePath.combine()`](combine/index.md)

    Combine file paths into one.

- [`FilePath.forceSlash()`](forceSlash/index.md)

    Converts all path separators to slashes.

- [`FilePath.forceBackslash()`](forceBackslash/index.md)

    Converts all path separators to backslashes.
