# Installer

The **`Installer`** interface is used to describe a MSI installer.

## Instance methods

- [`Installer.setInstallDir()`](setInstallDir/index.md)

    Sets the installation directory.

- [`Installer.setMsiProductName()`](setMsiProductName/index.md)

    Sets the MSI product name.

- [`Installer.setMsiProductVersion()`](setMsiProductVersion/index.md)

    Sets the MSI product version.

- [`Installer.setMsiManufacturer()`](setMsiManufacturer/index.md)

    Sets the MSI manufacturer.

- [`Installer.setMsiProductCode()`](setMsiProductCode/index.md)

    Sets the MSI product code.

- [`Installer.setMsiUpgradeCode()`](setMsiUpgradeCode/index.md)

    Sets the MSI upgrade code.

- [`Installer.setMsiPackageCode()`](setMsiPackageCode/index.md)

    Sets the MSI package code.

- [`Installer.addMsiProductKeywords()`](addMsiProductKeywords/index.md)

    Adds MSI product keywords.

- [`Installer.addComponent()`](addComponent/index.md)

    Adds a MSI component.

- [`Installer.addFeature()`](addFeature/index.md)

    Adds a MSI feature.
