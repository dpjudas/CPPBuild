# File: readAllBytes() static method

The **`readAllBytes()`** static method of the `File` object
reads an entire file from disk.

## Syntax

```js
File.readAllBytes(filename)
```

### Parameters

- `filename`

    A string with the path to the file.

### Return value

An array of bytes.

## Description

**`readAllBytes()`** reads an entire file from disk and returns it.
