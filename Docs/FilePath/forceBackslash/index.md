# FilePath: forceBackslash() static method

The **`forceBackslash()`** static method of the `FilePath` object replaces all slashes in a path with backslashes and returns the result.

## Syntax

```js
FilePath.forceBackslash(path)
```

### Parameters

- `path`

    A string with the file path.

### Return value

A string with no backslashes.

## Description

**`forceBackslash()`** replaces all slashes in a path with backslashes and returns the result.
